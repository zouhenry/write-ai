How should markdown be rendered in AI Chat?

The AI model already outputs markdown (bold, lists, headers). Right now it displays as raw text.

Current — Plain Text
Here are the key rules for **its** vs **it's**:

1. **it's** = "it is" or "it has" (contraction)
- Example: It's raining outside.

2. **its** = possessive pronoun
- Example: The dog wagged its tail.

## Quick trick
Try replacing with "it is" — if it makes sense, use it's.
