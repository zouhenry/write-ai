# Chat Conversation History Design

**Date:** 2026-05-09  
**Feature:** Multi-conversation sidebar for AI Chat tab

## Overview

Add a ChatGPT-style conversation sidebar to the AI Chat tab. Users can start new conversations, switch between past ones, and delete them. Conversations are stored in `localStorage`. Titles are auto-generated by the AI after the first exchange.

## Data Model

All conversations are stored in `localStorage` under key `grammarLlmConversations` as a JSON array. The active conversation ID is stored under `grammarLlmActiveConversationId`.

```json
[
  {
    "id": "uuid-v4",
    "title": "Difference between its and it's",
    "createdAt": 1706000000000,
    "messages": [
      { "role": "assistant", "content": "Hello! I'm your AI assistant..." },
      { "role": "user", "content": "Explain the difference between its and it's" },
      { "role": "assistant", "content": "Its is possessive..." }
    ]
  }
]
```

**Migration:** On first load, if `grammarLlmChatHistory` exists, convert it into the new format as the first conversation with title "Previous conversation", then delete the old key.

## Sidebar Component

**New Chat button** (top of sidebar):
- Creates a new conversation object: `{ id: uuid(), title: "New conversation", createdAt: Date.now(), messages: [] }`
- Appends to `grammarLlmConversations` in localStorage
- Sets it as the active conversation
- Clears the chat history view and re-renders the welcome message

**Conversation list:**
- Renders one item per conversation, most recent first
- Each item shows the title truncated to fit one line
- Active conversation is highlighted
- Each item has a đź—‘ delete button (visible on hover)
- Clicking an item switches to that conversation

**Switching conversations:**
- Saves current in-memory `chatHistory` to the current conversation in localStorage
- Loads the selected conversation's messages into `chatHistory`
- Re-renders the chat history view

## Title Generation

- After the first user message receives an AI response, fire a second `/chat` request with:
  - `message`: `"Summarize this conversation topic in 6 words or fewer."`
  - `history`: the first user message + AI response
- Update the conversation's `title` in localStorage and re-render the sidebar
- Until the title returns, display the first 40 characters of the user's first message as a placeholder title

## Delete

- Clicking đź—‘ removes the conversation from `grammarLlmConversations` in localStorage
- Re-renders the sidebar
- If the deleted conversation was active:
  - Switch to the most recent remaining conversation, OR
  - If none remain, create a fresh empty conversation and set it active
